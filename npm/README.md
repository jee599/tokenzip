# contextzip

Claude Code context optimizer. Compress CLI output by 60-90%.

## Install

```bash
npx contextzip
```

That's it. Restart Claude Code.

## What is this?

This is a thin installer for the [ContextZip](https://github.com/jee599/contextzip) Rust binary.
It downloads the pre-built binary for your platform and sets up the Claude Code hook.

See the [main repo](https://github.com/jee599/contextzip) for full documentation.
